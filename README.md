This is a app for exercise Graphql and Apollo Client.

## Getting Started

First, Develop Apps:

1. clone the repo
```bash
git clone: https://github.com/rwndy/exercise-graphql-nextjs.git
```
2. install packages
```bash
yarn 
```
3. Start this App:
```bash
yarn dev -p 3000 or you can change 3000 with another port
```

this app build with
  <ul>
    <li>NextJS</li>
    <li>chakra-ui</li>
    <li>Graphql and Apollo Client</li>
  </ul>
